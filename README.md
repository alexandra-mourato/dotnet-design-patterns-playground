# dotnet-design-patterns-playground
A .NET 8 playground demonstrating multiple GoF + enterprise patterns (Facade, Strategy, Decorator, Chain of Responsibility, State, Domain Events, Repository, Unit of Work, …)

## Project Structure

- `/src` – Production code
- `/tests` – Automated tests
- `/docs` – Architecture, patterns and decision logs
- `CheckoutPlayground.sln` – Solution file (root-level)
